# Run Liquid online
An online tool to run Liquid code snippet. - Build by [Bravo Yeung](https://geekplayers.com)

## Technology adopted
- liquidjs
- CodeMirror
- Liquid mode for Code Mirror
- jQuery

## Features
- Online highlight for liquid code
